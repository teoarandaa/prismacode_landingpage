Preguntas Frecuentes

Respuestas a las preguntas más comunes sobre nuestros servicios de desarrollo web y aplicaciones móviles.

¿Qué servicios ofrece Prisma Developments?

Ofrecemos una amplia gama de servicios digitales, incluyendo:

  • Desarrollo Web: Sitios corporativos, tiendas online (e-commerce), blogs, landing pages y aplicaciones web personalizadas.
  • Desarrollo de Aplicativos: Apps nativas para iOS y Android, así como soluciones multiplataforma con Flutter o otras tecnologias para desarrollos mas técnicos.
  • Soporte y Mantenimiento: Cambios menores, actualizaciones, corrección de errores, mejoras de rendimiento y seguridad para proyectos hechos por nosotros.
  • Consultoría Digital: Asesoramiento estratégico para la transformación digital de tu negocio.

¿Desarrollan aplicaciones para iOS y Android?

Sí, desarrollamos aplicaciones tanto para iOS como para Android. Podemos crear apps nativas específicas para cada plataforma utilizando Swift para iOS y Kotlin para Android, garantizando el mejor rendimiento y experiencia de usuario.

También ofrecemos desarrollo cross-platform con Flutter, que permite crear aplicaciones para ambos sistemas operativos a partir de un único código base, reduciendo tiempos y costes de desarrollo mientras se mantiene una alta calidad.

¿Puedo actualizar mi sitio web existente en lugar de crear uno nuevo?

Sí, ofrecemos servicios de rediseño y actualización para sitios web existentes. Podemos evaluar tu web actual e implementar mejoras en diseño, usabilidad, rendimiento, seguridad y SEO sin necesidad de empezar de cero.

Si tu web está construida con tecnologías obsoletas o presenta problemas estructurales graves, te asesoraremos sobre la mejor opción, ya sea una actualización o un rediseño completo.

¿Cuánto cuesta desarrollar una página web o aplicación?

El coste de desarrollo varía según la complejidad, funcionalidades y alcance del proyecto:

  • Sitios web: Desde 250€ para webs informativas básicas o más para proyectos complejos con e-commerce y funcionalidades personalizadas.
  • Aplicaciones móviles: Desde 500€ para apps sencillas o más para aplicaciones complejas con múltiples funcionalidades, integraciones y paneles de administración.
  • Soporte continuo: 384€ mensuales para mantenimiento, actualizaciones y mejoras continuas.

Para obtener un presupuesto personalizado, contáctanos detallando tu proyecto y requerimientos específicos.

¿Cuáles son las formas de pago aceptadas?

Aceptamos diferentes formas de pago para adaptarnos a tus necesidades:

  • Transferencia bancaria
  • PayPal

Trabajamos con un modelo de pagos por fases, normalmente:

  • 10% al inicio del proyecto
  • 20% al diseñar el wireframe
  • 40% al completar el desarrollo principal
  • 20% al acabar de pulir el proyecto
  • 10% antes de entregar el proyecto

¿Cuánto tiempo tarda en desarrollarse un proyecto?

Los tiempos de desarrollo dependen del tipo y complejidad del proyecto:

  • Sitios web informativos: Depende de la complejidad del proyecto, pero normalmente entre 2 y 4 semanas.
  • Aplicaciones web complejas: Depende de la complejidad del proyecto, pero normalmente entre 4 y 8 semanas.
  • Aplicaciones móviles: depende de la complejidad del proyecto, pero normalmente entre 6 y 12 semanas.

Estos plazos son orientativos y pueden variar según requerimientos específicos, cambios durante el desarrollo y rapidez en la revisión y aprobación por parte del cliente.

Siempre establecemos un cronograma detallado al inicio del proyecto para que conozcas los plazos exactos.

¿Cómo es el proceso de desarrollo?

Nuestro proceso de desarrollo sigue una metodología ágil que incluye las siguientes fases:

  1. Descubrimiento y planificación: Analizamos tus necesidades, objetivos y requisitos para definir el alcance del proyecto.
  2. Diseño UX/UI: Creamos wireframes y diseños visuales para definir la estructura y apariencia del producto.
  3. Desarrollo: Programamos las funcionalidades utilizando las tecnologías más adecuadas, dividiendo el trabajo en sprints.
  4. Pruebas y control de calidad: Realizamos pruebas exhaustivas para garantizar que todo funciona correctamente.
  5. Lanzamiento: Desplegamos la solución en los servidores de producción y la hacemos disponible para los usuarios.
  6. Soporte y mantenimiento: Ofrecemos soporte post-lanzamiento para resolver cualquier incidencia y mantener el producto actualizado.

Durante todo el proceso, mantenemos una comunicación constante contigo, con revisiones regulares y entregas incrementales para que puedas ver el avance del proyecto.

¿Ofrecen soporte después de finalizar el proyecto?

Sí, ofrecemos varios niveles de soporte post-lanzamiento:

  • Soporte básico: Incluido durante el primer año tras la finalización del proyecto, cubre la corrección de errores relacionados con la implementación original.
  • Soporte Continuo: Un servicio mensual a 384€ que incluye mantenimiento proactivo, actualizaciones de seguridad, pequeñas mejoras, monitorización de rendimiento y soporte prioritario.
  • Soporte puntual: Para clientes que prefieren solicitar asistencia específica solo cuando la necesitan, facturado por horas o por intervención.

Recomendamos el Soporte Continuo para mantener tu solución digital siempre actualizada, segura y con un rendimiento óptimo.

¿Qué tecnologías utilizan en sus desarrollos?

Utilizamos tecnologías modernas y contrastadas que garantizan soluciones robustas, escalables y de alto rendimiento:

  • Desarrollo Frontend: HTML5, CSS3, JavaScript, TypeScript, React, Angular, Next.js
  • Desarrollo Backend: PHP, Laravel, Python, Java
  • Desarrollo Móvil: Swift (iOS), Kotlin (Android), Flutter (multiplataforma)
  • Bases de datos: MySQL, PostgreSQL, MongoDB

Seleccionamos la pila tecnológica más adecuada para cada proyecto específico, considerando factores como requisitos funcionales, escalabilidad, presupuesto y plazos.

¿Puedo actualizar o ampliar mi proyecto en el futuro?

Absolutamente. Todos nuestros desarrollos están diseñados pensando en la escalabilidad y el crecimiento futuro:

  • Utilizamos arquitecturas modulares que facilitan la adición de nuevas funcionalidades.
  • Implementamos código limpio y bien documentado que permite futuras modificaciones.
  • Empleamos tecnologías con soporte a largo plazo para asegurar la viabilidad futura.

Muchos de nuestros clientes comienzan con un MVP (Producto Mínimo Viable) y van ampliando su solución a medida que su negocio crece o surgen nuevas necesidades.
